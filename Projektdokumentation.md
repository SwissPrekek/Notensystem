# Notensystem
## Funktionen
Das Notensystem welches von Fabian Zeller und Benjamin Girard programmiert wurde, enthält folgende Funktionen.
* Nutzer erstellen
* Nutzer einloggen
* Nutzer ausloggen
* Durchschnittsnoten des eingeloggten Users anzeigen
* Fächer anzeigen
* Fach hinzufügen
* Note / Prüfung hinzufügen
